# CareBridge iOS

CareBridge iOS is a SwiftUI app for childcare teams to manage attendance, daily diaries, incidents, communication, and child profile workflows.

## Modules
- **Core app**: app entry, navigation, and shared state
- **Models & sample data**: domain entities and local mock data
- **Services**: data management, attendance, messaging, notifications, sleep tracking
- **Utilities**: constants, validation, theming, haptics, extensions
- **Components**: reusable UI building blocks and widgets
- **Feature views**: splash, onboarding, dashboard, attendance, messages, daily diary, incidents, profile, checklist, notifications, settings
- **ViewModels**: dashboard, diary, and incident presentation logic
